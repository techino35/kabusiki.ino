# CATALYST ALPHA ENGINE

日本株のカタリスト投資判断エンジン。モンテカルロシミュレーションによるリスク分析付き。

## 概要

「AIに投資判断を任せたらどうなるか」を検証するプロジェクト。  
v1.0からv5.0まで段階的に現実の要素を追加し、10,000回のモンテカルロシミュレーションで統計的に検証した。

最終的に**AIの構造的限界**に到達し、エンジンの役割を「投資判断」から「NO-GO判定（やるべきでないトレードのフィルタリング）」に転換した。

## 主な発見

### 1. 行動バイアスの影響は想像以上に大きい

同じ市場・同じ銘柄・同じ戦略で、行動制御だけ変えた場合：

| 防御レベル | 1年後中央値 | 元本割れ確率 |
|-----------|-----------|-----------|
| Lv0（防御なし） | ¥7,153 (-28%) | 77.2% |
| Lv2（エンジン使用） | ¥12,462 (+25%) | 35.6% |
| Lv3（完全規律） | ¥14,148 (+41%) | 27.9% |

### 2. AIヘッジファンドはインデックスに負けている

Eurekahedge AI Hedge Fund Index（2009-2024）：年率9.8%  
同期間S&P500：年率13.7%

AIファンドの集合体がインデックスに負けている事実は、AIの構造的限界を示している。

### 3. AIの正しい使い方は「NO-GOマシン」

トップの投資ファンド（Bridgewater、Citadel、Point72）はAIを判断に使っていない。  
分析補助・執行・リスク管理に使い、最終判断は人間が行っている。
